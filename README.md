# 新南方职业培训学院自动签到脚本

## 📋 项目简介

自己老是忘记签到...所以空闲时间整了个签到脚本

这是一个为新南方职业培训学院网站（`jxedu.xnfedu.com`）开发的自动签到脚本，专为需要完成继续教育学时的专业技术人员设计，用于专业课学时。脚本可自动登录、执行签到并查询红豆数量，解决因工作繁忙而忘记签到导致学时不足的问题。

> **应用场景**：适用于教师、医生、工程师等需要完成继续教育专业课学时要求的专业人员。

---

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| ✅ 多账号支持 | 支持同时配置多个账号，一次运行批量签到 |
| ✅ 自动登录 | 模拟真实登录流程，处理ASP.NET表单验证 |
| ✅ 智能签到 | 自动检测今日是否已签到，避免重复操作 |
| ✅ 进度查询 | 签到后自动查询剩余红豆和累计红豆数量 |
| ✅ 结果汇总 | 清晰展示每个账号的签到结果和红豆变化 |
| ✅ 本地运行 | 完全在本地执行，无需担心账号泄露风险 |

## 🚀 快速开始

### 环境要求
- Python 3.6+
- 依赖库：`requests`

### 安装依赖
在运行脚本前，请确保电脑已经安装了python，且配置好了环境变量，然后cmd执行下面命令：
```bash
pip install requests
```
### 配置说明
- 下载脚本文件。
- 使用文本编辑器打开脚本文件。
- 在文件开头的 ACCOUNTS配置区域，按以下格式填写账号信息：
```python
ACCOUNTS = [
    {"username": "账号1", "password": "密码1"},
    {"username": "账号2", "password": "密码2"},
    # 可继续添加更多账号
]
```

### 使用方法
- 完成配置后，在终端（命令行）中运行以下命令即可启动自动签到：
```bash
python xnfedu_checkin.py
```
- 或者直接右键打开脚本执行

## 🕐 配置自动化任务（定时运行）
**为了彻底解放双手，实现每日自动签到，可以使用 Windows 系统自带的 “任务计划程序”​ 将此脚本配置为定时任务。**  
（别问为什么只给Windows教程，你用Linux要是不会设定时任务别玩了...）  

**操作步骤**
1. **打开任务计划程序**
    *   按下键盘上的 `Win + R` 键，打开“运行”对话框。  
    *   输入 `taskschd.msc` 并回车，即可打开“任务计划程序”窗口。  
2. **创建基本任务**
    *   在右侧的 **“操作”** 侧边栏中，点击 **“创建基本任务”**。  
    *   在弹出的向导中：  
           名称：输入一个易于识别的任务名称  
           描述：（可选）输入对此任务的简单描述。
           点击 “下一步”。
3. **设置触发器（何时运行）**
    *   选择 “每天”， 点击 “下一步”。
    *   设置你希望每日执行签到的时间（例如：8:30:00）， 点击 “下一步”。
4. **设置操作（运行什么）**
    *   选择 “启动程序”，点击 “下一步”。
    *   选择程序或脚本：点击 “浏览”，找到并选择你电脑上的脚本文件，点击 “下一步”，完成。
5. **测试**
    *   任务创建后，你可以在任务计划程序库中看到它。可以右键点击该任务，选择 **“运行”** 来立即测试一次，确保配置正确。

## 注意事项
- 账号安全：账号密码以明文形式存储在本地脚本中，请务必妥善保管脚本文件，避免泄露。
- 兼容性：本脚本依赖于目标网站当前的HTML结构。若网站前端进行重大改版，可能导致脚本失效。
- 使用规范：请合理使用脚本，本工具旨在帮助因工作繁忙可能遗忘操作的用户，请勿用于任何盈利或干扰平台正常运行的用途。
