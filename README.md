# 新南方职业培训学院自动签到脚本

## 📋 项目简介

这是一个为新南方职业培训学院网站（`jxedu.xnfedu.com`）开发的自动签到脚本，专为需要完成继续教育学时的专业技术人员设计。脚本可自动登录、执行签到并查询学习进度（红豆数量），解决因工作繁忙而忘记签到导致学时不足的问题。

> **应用场景**：适用于教师、医生、工程师等需要完成继续教育专业课学时要求的专业人员。

---

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| ✅ 多账号支持 | 支持同时配置多个账号，一次运行批量签到 |
| ✅ 自动登录 | 模拟真实浏览器登录流程，处理ASP.NET表单验证 |
| ✅ 智能签到 | 自动检测今日是否已签到，避免重复操作 |
| ✅ 进度查询 | 签到后自动查询剩余红豆和累计红豆数量 |
| ✅ 结果汇总 | 清晰展示每个账号的签到结果和红豆变化 |
| ✅ 本地运行 | 完全在本地执行，无需担心账号泄露风险 |
